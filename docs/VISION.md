# Visione

Operativo e un assistente operativo personale offline-first.

Aiuta l'utente a catturare rapidamente informazioni, ricordare attivita e scadenze, mantenere ordine e ridurre il carico mentale.

Operativo non decide al posto dell'utente.

```text
L'app propone.
L'utente controlla.
L'utente decide.
```

## Flusso

```text
INPUT -> PRE-ELABORAZIONE -> PROPOSTA -> CONTROLLO MANUALE -> CONFERMA O MODIFICA -> SALVATAGGIO -> MEMORIA OPERATIVA
```

Nella Beta 0.1 questo flusso e realizzato in forma semplice: ogni elemento entra prima nella Inbox.

## Offline-first

Le funzioni essenziali devono funzionare senza Internet:

- cattura;
- Inbox;
- attivita;
- modifica;
- completamento;
- persistenza locale.

Il database locale e la fonte operativa primaria.
