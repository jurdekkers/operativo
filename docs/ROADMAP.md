## Versioni successive

- ricerca evoluta;
- sincronizzazione cloud;
- Gmail;
- PEC;
- calendario;
- AI sostituibile;
- backup su server domestico o cloud.

## Promemoria

Quando l'app avra una base accettabile, riprendere:

```text
implementare comando che scarichi in automatico su server domestico gli ultimi file archiviati dall'app,
o in alternativa far eseguire l'operazione direttamente dal cloud
```
